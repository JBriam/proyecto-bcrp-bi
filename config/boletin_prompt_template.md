# Plantilla de Prompt para Boletin BCRP

## Instrucciones
Usa el JSON generado en `data/processed/boletin_input.json` como entrada principal.

## Prompt base

Eres un analista economico senior. Con base en el JSON adjunto, redacta un boletin ejecutivo sobre la economia del Peru.

Requisitos:
- Escribe en espanol claro y profesional.
- Resume las series por tema: inflacion, politica monetaria, tipo de cambio, actividad, empleo y sector externo.
- Destaca variaciones relevantes y cambios recientes de tendencia.
- Indica la lectura ejecutiva de forma breve y accionable.
- Si hay valores atipicos o faltantes, menciónalos con cautela.

Estructura sugerida:
1. Resumen ejecutivo.
2. Inflacion y politica monetaria.
3. Actividad y empleo.
4. Sector externo.
5. Conclusiones.

Estilo:
- Prioriza claridad y sintesis.
- Evita lenguaje tecnico innecesario.
- Usa datos concretos cuando existan.
